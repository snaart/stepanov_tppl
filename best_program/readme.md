# Какой-нибудь заголовок про best program

Сверхнадежная программа для сбора данных с нащих нестабильных серверов

## Особенности

- **Гарантированная запись:** Использует `fsync`/`O_DSYNC` после каждой строки. Данные сохраняются даже при выдергивании шнура
- **Отказоустойчивость:** Автоматическое переподключение при разрыве сети, таймаутах или падении сервера
- **Многопоточность:** Независимая обработка каждого источника данных
- **Качество кода:** Покрытие Unit-тестами (Google Test) + Интеграционные тесты

## Требования

- C++17 компилятор (GCC/Clang)
- CMake 3.14+
- Python 3 (для интеграционных тестов)
- Интернет (для автоматического скачивания GoogleTest при сборке)
